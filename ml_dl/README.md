# 📈 Machine Learning & Deep Learning (ML/DL)

This directory is dedicated to classical machine learning and deep learning applications using the MedBridge clinical datasets.

## 🔬 Focus Areas

### 1. Traditional ML
-   **Patient Classification**: Grouping patients by risk level using `scikit-learn`.
-   **Feature Engineering**: Extracting meaningful signals from lab events and diagnoses.

### 2. Deep Learning & GNNs
-   **Graph Neural Networks**: Using `PyG` (PyTorch Geometric) to predict links in the clinical Knowledge Graph.
-   **Sequence Modeling**: Using RNNs/LSTMs or Transformers to model longitudinal patient journeys.

## 🧪 Getting Started

Explore the `scripts/` directory for data loading utilities and refer to the root `README.md` for project ideas. Stay tuned for baseline notebooks!
